@classmethod

基本介绍

  • @classmethod 是 Python 中的一个装饰器，作用是把一个方法封装成类方法 。

  • 类方法是什么，就是可以直接使用类，无需实例化调用的方法叫做类方法。

  • 比如:

  • class DateUtils:    
    @classmethod
    def get_time(cls):
        import datetime
        return datetime.datetime.now()
 
#类调用
year1 = DateUtils.get_time
print(year1)

#实例化后调用
date = DateUtils()
year2 =n date.get_time
print(year2)
