  • num_rows 功能解析

    在 PHP 的 mysqli 扩展中，当你执行一个 SELECT 类型的 SQL 查询语句后，会得到一个 mysqli_result 对象（即代码中的 $result ） 。num_rows 是 mysqli_result 对象的一个属性，它用于获取查询结果集中的行数。

    这里通过 $result->num_rows 获取到结果集的行数，从而可以判断是否有数据被查询出来，方便后续对结果进行处理。如果执行的查询没有匹配到任何行，num_rows 的值为 0 ；如果有匹配的行，它的值就是匹配行的数量。

    fetch_assoc 功能解析

    fetch_assoc 是 mysqli_result 对象的一个方法。它的作用是从结果集中获取一行数据，并将其作为关联数组返回。关联数组的键是结果集中列的名称，值是对应列的值。

  • fetch_assoc()：

    • 返回 关联数组，键为列名（如 'name'、'points'）。
    • 只能通过列名访问数据（$row['name']）。

  • fetch_row()：

    • 返回 索引数组，键为数字索引（如 [0]、[1]）。
    • 只能通过索引访问数据（$row[0]）。

  • fetch_array()：

    • 返回 混合数组，同时包含关联键和索引键。
    • 可以通过列名（$row['name']）或索引（$row[0]）访问数据。
